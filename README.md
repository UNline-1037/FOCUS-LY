# FOCUS-LY
# 主要用于FOCUS2026年春招
# 功能有：
# 1. Python 后端服务
# 2. HTML/JS 前端⻚面 
# 另外有指导文档LEARNING_log.md

介绍：
requirements.txt内有所需开发环境依赖包，下载后只要pip install -r requirements.txt即可安装
static内有所需的CSS模板，JavaScript文件以及所用到的图片
templates里有网页.html文件，用浏览器打开即可看见
test1.py为后端代码，打开网页前需要先运行

注意事项：
要保证电脑上的Github流畅，否则会影响网页的打开！！
